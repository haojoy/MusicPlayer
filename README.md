# MusicPlayer
音乐播放器 **musicplayer for learning qt**

环境：win10 、QT 5.15.2、Qt Creator

#### 当前主要功能：

歌曲在线搜索及搜索结果展示

歌词及封面展示 （UI参考酷狗音乐）

歌曲播放：播放模式选择 音乐音量控制 进度条控制 （UI参考酷狗音乐）

我的喜欢 ：添加收藏

最近播放 

本地音乐

创建歌单 

桌面歌词

背景设置：单独设置背景颜色与背景图片 (UI参考网易云音乐）

其它功能待完善...

#### 说明：

图标来源 https://www.iconfont.cn/

实用工具：取色器takecolor
